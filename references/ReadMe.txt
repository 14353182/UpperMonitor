************************** ZM124xUE系列接口函数动态链接库文件 **********************

	1. ZM124U.dll 	动态链接DLL程序
	2. ZM124U.h		ZM124xUE系列接口头文件声明
	3. ZM124U.lib	动态链接DLL程序引入库文件

************************** ZM124xUE系列接口函数动态链接库文件 **********************



************************************* 使用说明 *************************************

	1. 在工程目录下新建libs文件夹用于存放第三方相关库文件；
	2. 将编译链接所需的引入库文件ZM124U.lib还有头文件声明ZM124U.h拷贝到./libs下
	3. 在使用ZM124xUE系列接口的源文件中包含头文件：
		#include "./libs/ZM124U.h"
	   其中./表示工程目录，之后源文件便能编译通过，但是Link时会出错，因为找不到具体实现；
	4. 通过引入库文件解决链接出错的问题：
		第一种： 在上述#include语句前一行添加#pragma comment(lib,"./libs/ZM124U.lib")预编译头，
			链接时引入该引入库文件；
		第二种： Project->Settings->Link，Category选择Input表示输入项，Object/library modeules输入ZM124U.lib，
			Additional library path输入./libs表示工程目录下的libs库文件目录。
	5. 最后，需要把DLL动态链接库文件与Debug目录下的可执行.EXE程序放在同一目录下，不然运行时会因搜索不到对应的DLL文件，导致程序无法执行。
